# Closing Remarks

A mobile-friendly real-estate dad-joke generator. It includes 50 jokes, shuffles them into a local deck, and avoids repeats until the full collection has been shown.

## Run locally

Open `index.html` directly, or start any static file server in this directory.

## Publish free

Upload the four site files to GitHub Pages, Netlify Drop, or Vercel. No server, account database, or API key is required.
